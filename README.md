<h3>¿Cómo instalar?</h3>
Descargue e instale la versión más reciente de XAMPP de https://www.apachefriends.org/index.html, siga las instrucciones en https://blogs.oracle.com/opal/installing-xampp-for-php-and-oracle-database y https://www.youtube.com/watch?v=U27xP9bRs0w. Descargue este proyecto. Corra el script de creacion de usuario en la base de datos Oracle, seguido del script hospital_admin.sql. Guarde la carpeta de Proyecto_Lenguajes_BD en C:/xampp/htdocs/, inicie el servicio de Apache en XAMPP y acceda a http://localhost/Proyecto_Lenguajes_BD en su navegador.
